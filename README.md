# Как подставить свой вариант, если немножко не знамо, куда жмать?

* Консольное приложение:
    1. Заходите в папку с тем же названием, что и у решения – файл .sln.
    2. Откройте файл Program.cs.
    3. Там, где указаны комменты, заменяете значение на своё из варианта. Для тех, кто не знает, что такое комменты – это слова после двойного слэша (//) или в такой конструкции – /**текст**/.
    
    После этих действий, необходимо пересобрать решение, чтобы отобразился именно ваш вариант.
    
    Если у вас есть Visual Studio (не VS Code) и вы редачите через него, то через него пересоберите решение и запустите как в Debug, так и в Release.
    
    Если у вас нет Visual Studio и вы чуток не знаете, как пересобирать решения на C#, то проверяем, есть ли у вас .net sdk. 
    Если нет, то скачиваем его (VS жрет память). Через консольную строку вбиваем следующие команды:
    1. cd /**путь/ к/ папке/ с/ файлом/ .sln**/
    2. dotnet clean
    3. dotnet build --configuration Debug
    4. dotnet build --configuration Release

* WinForms:
    1. Повторяете пункт 1 из консольного приложения.
    2. Откройте файл MainForm.Designer.cs. Здесь хранится шаблон самой формы, которая будет открываться при запуске программы.
    3. Ищите следующие элементы через ctrl+f: StudyingDaysBox, BirthHourBox, LastNameBox, BirthDatePicker.
    4. Там, где увидите комментарии, подставляем следующее и заменяем на своё, что хотите:
          LastNameBox.Text = "Укажи свою фамилию";
          BirthDatePicker.Value = new DateTime(Твой год рождения, Твой месяц рождения, Твой день рождения, 0, 0, 0, 0);
          StudyingDaysBox.Text = "Введи рандомное число";
          BirthHourBox.Text = "Введи рандомное число";
    !! ВАЖНО. Подставляем строки на место тех строк, после которых написан комментарий в духе //Меняем бла-бла-бла
    !! Поломаешь, будешь сам/а чинить!

    Всё то, что вы поменяли, будет отображаться в окне запущенного приложения. Здесь мы меняем только значения, которые будут отображаться по умолчанию.
    
    После редактирования также потребуется пересобрать решение, чтобы отобразился именно ваш вариант.

Когда всё будет у Вас готово, для проверки можете запустить ярлык приложения, который лежит вместе с файлом решения. Шарящие сами знают, как запустить прогу без ярлыка :) 
Если там отображаются ваши изменения, то поздравляю! Вы теперь шкодер!
